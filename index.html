    วิธีใช้งาน (อ่านก่อนรัน):
    1. เข้าสู่ระบบ Supabase แล้วเข้าโปรเจกต์ของคุณ
    2. ไปที่เมนู SQL Editor (ไอคอนรูป 📝 < > แถบซ้ายมือ) > กดปุ่ม New query
    3. กดปุ่ม คัดลอกโค้ด ด้านล่าง แล้วนำไปวางในช่องว่างของ SQL Editor
    4. กดปุ่ม Run (ปุ่มสีเขียวมุมขวาล่าง) รอจนขึ้นคำว่า Success
    5. สำคัญมาก! หลังจากรันเสร็จ ให้ไปที่เมนู Authentication > Policies แล้วไล่กด Disable RLS ให้ครบทั้ง 10 ตาราง
